You may need a tool to perform a specific function during your project. This isn't an exhaustive list but it is a list of things that I've found useful in previous projects. Don't assume you have to use things from this list and you should do some research for yourself into tools that may be useful. These are in no particular order so you may have to read the entire list to find things that are useful for you.

Step Recorder - Records the process you sent through in order to perform a task. Useful so that you can look back and see what you did when configuring something. It will create a zip file, inside is a MHTML which you can look at in IE.
